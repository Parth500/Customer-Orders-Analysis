# Customer Orders Analysis

## Project Overview

This project analyzes customer order data using Python, Pandas, and Matplotlib.

## Technologies Used

- Python
- Pandas
- Matplotlib
- JupyterLab

## Dataset

The dataset includes:

- Order ID
- Customer
- Category
- Sales
- Quantity
- Discount

## Analysis Performed

- Data loading
- Missing value handling
- Feature engineering
- Revenue calculation
- Category analysis
- Customer analysis
- Data visualization
- Business insights

## Key Features

- Cleaned missing values
- Created a new `Final_Sales` column
- Identified top customers
- Calculated business KPIs
- Generated charts for reporting
